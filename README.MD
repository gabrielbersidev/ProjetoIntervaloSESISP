🍎 Projeto Merenda Consciente
Este foi o meu primeiro projeto colaborativo, desenvolvido quando eu tinha 13 anos. Ele nasceu da vontade de resolver um problema real na minha escola: o desperdício excessivo de comida e, simultaneamente, a falta de lanche para alguns alunos.

A proposta é um sistema de gerenciamento onde alunos registram sua intenção de lanche, gerando dados em tempo real para a equipe da cozinha.

🤝 Colaboração Internacional
O projeto foi desenvolvido em parceria com um amigo estudante da University of Wisconsin-Madison. Foi minha primeira experiência compartilhando um repositório, lidando com revisões de código e colaborando de forma remota, o que tornou o aprendizado muito mais rico.

🚀 Tecnologias Utilizadas
Para construir essa solução, utilizamos uma stack moderna de desenvolvimento Web:

Frontend: HTML5 e CSS3 (Interface focada em usabilidade para alunos e visibilidade na TV).

Backend: Node.js (Processamento de login e lógica de negócios).

Banco de Dados: MongoDB (Armazenamento persistente das confirmações e perfis dos alunos).

📋 Como Funciona?
Login do Aluno: O aluno acessa a interface (index) e faz sua autenticação.

A Escolha: Uma interface simples onde ele clica se irá ou não consumir o lanche no dia.

Dashboard da Cozinha: Uma página dedicada para exibição em uma Smart TV na cozinha, mostrando para as nutricionistas e tias da cozinha o número exato de porções a serem preparadas.

Benefícios Alcançados:
Zero Desperdício: Produção baseada em dados, não em estimativas.

Segurança Alimentar: Garantia de que nenhum aluno que confirmou ficará sem o lanche.
